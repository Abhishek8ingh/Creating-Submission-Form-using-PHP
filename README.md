# 🌍 Travel Form Web Application (PHP + MySQL)

## 📌 Project Overview

This is a simple **Travel Form Web Application** built using **PHP, MySQL, HTML, and CSS**.
Users can enter their details through a form, and the data is stored in a MySQL database.

---

## 🚀 Features

* User-friendly travel registration form
* Data stored securely in MySQL database
* Backend processing using PHP
* Clean and responsive UI using HTML & CSS
* Success message displayed after form submission

---

## 🛠️ Technologies Used

* **Frontend:** HTML, CSS  
* **Backend:** PHP 
* **Database:** MySQL (XAMPP)

---

## 📂 Project Structure

```
/project-folder
│── index.php        # Main backend + frontend logic
│── style.css        # Styling file
│── index.html       # Initial frontend structure 
│── images           # Background image, SS of WebPage
```

---

## ⚙️ How It Works

1. User fills out the travel form
2. Form sends data using POST method
3. PHP processes the data
4. Data is inserted into MySQL database
5. Success message is displayed

---

## 🧠 Key Concepts Used

* Client-Server Architecture
* HTTP POST Method
* PHP Form Handling (`$_POST`)
* MySQL Database Connection
* SQL INSERT Query

---

## 📸 Output

A travel form where users can:

* Enter name, age, gender, email, phone
* Submit details
* Data gets stored in database
